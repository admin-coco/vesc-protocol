Li.Fi Swap — 2 Gaps to Know:   (1) Solana is not supported by Li.Fi — Li.Fi only routes across EVM chains. Solana users cannot swap to/from other chains via Li.Fi. You will need a separate swap solution for Solana (e.g. Jupiter, direct bridge).   (2) VESC (Base) is not listed on Li.Fi — VESC is a custom token. It will not appear in Li.Fi's token registry unless manually added or whitelisted. Users cannot swap VESC via Li.Fi until this is resolved.
Venezuela: Only Crixto is SUNACRIP-licensed. All others (incl. El Dorado) block VE residents. Integrate Crixto as a separate flow for VE users.  |  Bridge.xyz is B2B API only — requires a separate Stripe agreement, not a drop-in widget.  |  CryptoBuyer is internal agreement only — no public docs. On-ramp for VESC on Base only.  |  Red WETH tokens (Polygon, BNB Chain) are wrapped ETH — not native. Use the WETH contract address from the dev reference below, not the native ETH zero address. ETH does not exist on Solana and has been removed.
